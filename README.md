# CenteNVIM

This is CentenV's Neovim configuration

## Additional Requirements

### Treesitter
For the latest version of `nvim-treesitter` to work, the `tree-sitter` CLI must be installed. See [tree-sitter](https://tree-sitter.github.io/tree-sitter/)

Because other plugins depend on Rust. We can run the following command to globally install `tree-sitter-cli` via `cargo`
```
cargo install tree-sitter-cli
```
